Nofollow

A value that can be assigned to a hyperlink to instruct search engines not to follow the link.

Description

Nofollow is an HTML attribute that can be added to links to indicate to search engines that they should not follow the link or pass any SEO credit to the linked page. This attribute is particularly useful in contexts where a website owner wants to avoid endorsing the linked content, such as in user-generated content, comments, or advertisements. By using the nofollow attribute, webmasters can maintain control over their site's link profile and prevent spammy links from affecting their site's search engine ranking. The nofollow attribute is added to the anchor tag in the following way: link text. Since its introduction by Google in 2005, the nofollow attribute has been widely adopted across the web, making it a standard practice for managing how links are treated by search engines. It is important to note that while nofollow links do not pass PageRank, they can still drive traffic and contribute to a site's visibility.

Examples

  • A blog uses nofollow on links in the comments section to prevent spam links from affecting SEO.
  • An online forum applies nofollow to user-generated posts to avoid endorsing potentially harmful or unverified content.

Additional Information

  • Nofollow links can still be clicked by users and can drive traffic to the linked site.
  • Search engines may still crawl nofollow links, but they won't attribute any link equity to them.
